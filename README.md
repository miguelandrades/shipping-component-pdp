# shipping-component-pdp
Este projeto consiste no desenvolvimento de um componente personalizado de cálculo de frete integrado à Página de Produto (PDP) dentro do ecossistema VTEX IO. A solução foi projetada para otimizar a experiência de compra, focando na redução da taxa de abandono ao fornecer clareza imediata sobre valores e prazos de entrega para o usuário final.
